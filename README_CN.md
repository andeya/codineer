Codineer 是一个**本地优先的编程智能体**，完全在你的终端中运行。它能读取你的工作区、理解项目结构，帮你编写、重构、调试和交付代码 — 支持交互式对话和一次性命令两种模式。

使用安全 Rust 构建，编译为单个独立二进制文件。无守护进程，无云端依赖（自带 API Key 即可）。

## 为什么选择 Codineer？

- **隐私优先** — 代码始终留在本地，只有你主动发送的提示词才会离开终端
